# Typing Speed Calculator

A simple C program to test your typing speed and accuracy. The program generates a random sentence of 20 unique words and gives you 60 seconds to type it as quickly and accurately as possible.

## Features
- Randomly generated sentence from a large English word bank
- 20 unique words per test
- 60-second timer
- Calculates words per minute (WPM) and accuracy
- Friendly feedback if time runs out

## How to Build and Run

1. Open a terminal in the project directory.
2. Compile the program using MinGW GCC:

```
c:/MinGW/bin/gcc.exe typing_test.c -o typing_test.exe
```
c:/MinGW/bin/gcc.exe 20wordtest.c -o 20wordtest.exe
3. Run the program:

```
.\typing_test.exe
```

## Usage
- The program will display a random sentence of 20 words.
- Press ENTER when ready, then type the sentence as fast and accurately as you can.
- If you finish before 60 seconds, your WPM and accuracy will be displayed.
- If time runs out, you'll be prompted to try again.

## Requirements
- Windows OS
- MinGW GCC (for compilation)
